Modular Garden System

2022

A custom PCB I designed in the KiCad software to fit the needs of my project which I also milled myself in FR-1 one-sided PCB on a Bantom milling machine.

Strawberries grow out of a garden unit banking up against the acrylic panneling within the lasercut frame of the garden. The inside is fully insulated and water tight.

The Unity program I designed and constructed to display sensor information to the user about the garden as well as allowing the user to control the pump.

What does the Modular Garden System do?

The Modular Garden System is a specific set of what are called Units, including the following: Control Unit, Water Management Unit, and Garden Unit. When constructed together they create a fully self-managing garden system, watering itself while also gathering data from the Garden(s) Unit via sensors. It connects via USB to any computer and communicates via USB Serial communication to a Unity-made program which displays sensor data and allows control of the water system through unique 3D visuals. The Units are constructed intentionally as hexagons to add the modularity, as Units snap together to form any shape or size that the user desires. The water system and piping system also follows that modularity and is constructed similarly.

Modularity & Expandability

As mentioned the system is delibratly designed in a hexagonal shape (all of the unnits) as well as the watering system so that the overall garden can be expanded to any shape or size that the user wants. The Units snap together and the water system includes six way intersections above each unit in order to deliver water to all connected Units..

Self-Watering Capabilities & Soil Moisture Detection

The Modular Garden System is equipted with a soil moisture sensor that detects the moisture of the soil of the nearest Garden Unit. Once the system detects too much of a lack of water, it will send serial data to the pump to begin pumping water into the system until the soil moisture sensor reads a high enough moisture level.

Serial Connection to Unity & the Unity Program

The Control Unit, which houses all of the chips, electronics, and sensors periodically sends serial data through an FTDI board to a USB cable to whatever computer may be connected to the system. The Unity program I developed connects to that USB port serial stream and reads the data in order to dispay it in an easily readable comprehensive way for the user. From the Unity program, the user can also send data back to the garden such as the ability to manually queue the pump from the program, allowing the user full control over the system.
